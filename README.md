# Typeracer-game

libstdc++-6.dll file is also needed to run the game - having issues trying to push the file into the git repository - may do later

Typeracer game that give your WPM, Accuracy, Correct and Incorrect characters.
Comes with 5 preinstalled texts to play through, and a way to input your own texts to play.

Game was made for a school project

User manual/Instructions:
To run the program, first extract all the files from the "Typeracer.zip" file anywhere.
Once done, run the exe file in either administrator or non-administrator mode.
This should then open a command prompt window, where the game will welcome you, then state the rules of the game.

Once this has been shown, you will be able to play the preset texts, or adding your own text and play it.
After playing, stats will be shown from the "run", after which you will be prompted to play again or not.
